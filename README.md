# How to build project
Have NodeJS and MongoDB installed then

Start MongoDB server.
```
mongod
```
<br>
Start dev server.
```
npm install
gulp serve
```
<br>
Start auth server.
```
node ./backend/server.js
```
<br>
visit localhost:3000
